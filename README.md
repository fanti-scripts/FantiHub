📱 Поддержка телефонов
Скрипт адаптирован для Delta, Arceus X и других инжекторов на мобильных устройствах.

🎯 Функции
Функция	Описание
Aimbot	Автонаведение на цель с настройкой FOV и плавности
ESP	Подсветка игроков (зелёный — мирный, синий — шериф, красный — убийца)
Stats	FPS + количество игроков на сервере
Teleport Tool	Инструмент для телепортации в выбранное место
Движение кнопки	На ПК и телефоне (пальцем)
Открытие меню	Правый Shift
🖥️ Управление
Действие	Как
Открыть меню	Правый Shift
Перетащить кнопку	Зажать ЛКМ (ПК) / Палец (телефон)
FOV настройки (ПК)	Правый клик по "Aimbot"
FOV настройки (телефон)	Долгий клик по "Aimbot" (0.5 сек)
Stats Select (ПК)	Правый клик по "Stats"
Stats Select (телефон)	Долгий клик по "Stats" (0.5 сек)
